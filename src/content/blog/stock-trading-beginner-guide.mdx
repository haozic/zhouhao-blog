入市前最重要的不是学会怎么赚钱，而是先学会怎么不亏钱。

## 一、看盘基础：如何读懂股票行情

### 1.1 K线基础

**K线**（Candlestick Chart，又称蜡烛图）是最常用的股票技术分析工具，由**开盘价、收盘价、最高价、最低价**四个要素构成，因形似蜡烛而得名。

**K线的构成：**
- **阳线（红色/实心）**：收盘价 > 开盘价，表示上涨
- **阴线（绿色/空心）**：收盘价 < 开盘价，表示下跌
- **上下影线**：反映盘中价格的波动范围

<Aside type='note' title='K线构成示意' style='card'>

```
     上影线
     ┌─┐
    ┌┘ └┐  ← 上影线最高点
  ──┤    ├──  ← 实体（开盘/收盘）
    └─┬─┘
     下影线
```

</Aside>

**常见K线形态：**

| 形态 | 特征 | 信号意义 |
|------|------|----------|
| 锤子线 | 下影线是实体的2-3倍，出现在低位 | 可能的见底信号 |
| 上吊线 | 与锤子线相反，出现在高位 | 可能的见顶信号 |
| 吞没形态 | 第二根K线完全吞没第一根 | 趋势反转信号 |
| 十字星 | 开盘=收盘，上下影线相当 | 观望信号 |

### 1.2 均线系统

**均线**（MA，Moving Average，移动平均线）是最基础的趋势判断工具，通过计算一定周期内的收盘价平均值并连线来反映价格趋势。

**常用均线周期：**

| 均线 | 周期 | 分析意义 |
|------|------|----------|
| MA5 | 5日 | 反映短期趋势（也称"周线"） |
| MA20 | 20日 | 反映中期趋势（也称"月线"） |
| MA60 | 60日 | 反映中长期趋势（也称"季线"） |
| MA120 | 120日 | 反映长期趋势（也称"半年线"） |

**均线使用核心原则：**

<Aside type='tip' title='多头排列与空头排列' style='card'>

**多头排列（强势信号）：**
MA5 > MA20 > MA60 > MA120（短期均线在长期均线上方）

**空头排列（弱势信号）：**
MA5 < MA20 < MA60 < MA120（短期均线在长期均线下方）

</Aside>

**均线交叉信号：**
- **黄金交叉**：短期均线上穿长期均线 → 买入信号
- **死亡交叉**：短期均线下穿长期均线 → 卖出信号

### 1.3 成交量解读

成交量是股价变动的动力来源，技术分析有句名言：**"量在价先"**——量的变化往往先于价格变化。

**量价关系四大原则：**

| 量价关系 | 市场含义 |
|----------|----------|
| 价升量增 | 健康的多头趋势，上涨有动力 |
| 价升量缩 | 顶背离，可能上涨乏力 |
| 价跌量增 | 抛压较大，可能继续下跌 |
| 价跌量缩 | 底背离，可能即将企稳 |

### 1.4 常用技术指标

#### MACD（指数平滑异同移动平均线）

**MACD**（Moving Average Convergence Divergence，指数平滑异同移动平均线）是最常用的趋势判断指标，由**DIF线**、**DEA线**和**MACD柱**组成。

- **DIF线**（Difference，正差离值）= EMA(收盘价, 12) - EMA(收盘价, 26)
- **DEA线**（Signal，信号线）= EMA(DIF, 9)
- **MACD柱** = 2 × (DIF - DEA)

<Aside type='tip' title='MACD交易信号' style='card'>

- **金叉**：DIF上穿DEA + MACD柱由绿转红 → 买入
- **死叉**：DIF下穿DEA + MACD柱由红转绿 → 卖出
- **背离**：价格创新低但MACD没有创新低 → 可能见底

</Aside>

#### KDJ随机指标

**KDJ**（Stochastic Oscillator，随机指标）是一种超买超卖指标，数值在0-100之间波动。

| 区域 | KDJ数值 | 信号 |
|------|---------|------|
| 超卖区 | J < 20 或 K、D < 20 | 可能是买入机会 |
| 超买区 | J > 80 或 K、D > 80 | 可能是卖出机会 |
| 中性区 | 20 - 80之间 | 观望 |

#### 布林带

**布林带**（Bollinger Bands，布林线）由三条线组成：
- **中轨**：20日均线
- **上轨**：中轨 + 2倍标准差
- **下轨**：中轨 - 2倍标准差

<Aside type='note' title='布林带使用要点' style='card'>

股价触及下轨可能获得支撑，触及上轨可能承受压力。布林带收窄往往预示行情即将突破。

</Aside>

## 二、交易策略入门：找到适合自己的方法

### 2.1 三大策略类型对比

| 策略类型 | 核心理念 | 持仓周期 | 适合人群 |
|----------|----------|----------|----------|
| 趋势跟踪 | 顺势而为 | 几周至几月 | 有时间看盘、能承受回撤 |
| 价值投资 | 买入优质公司 | 几个月至几年 | 有耐心、注重基本面 |
| 短线交易 | 利用日内波动 | 分钟至几天 | 有时间盯盘、能严格止损 |

### 2.2 趋势跟踪策略

**代表人物：** 理查德·丹尼斯（海龟交易法则创始人）

**海龟交易核心规则：**
1. **入场**：价格突破20日最高价时买入
2. **出场**：价格跌破10日最低价时卖出
3. **仓位**：每笔交易风险不超过总资金的**2%**
4. **品种分散**：同时跟踪多个不相关品种

### 2.3 价值投资策略

**代表人物：** 沃伦·巴菲特

**价值投资三大指标：**

#### 市盈率（P/E）

**P/E**（Price to Earnings Ratio，市盈率）= 股价 / 每股收益

| P/E范围 | 市场判断 |
|---------|----------|
| P/E < 0 | 公司亏损 |
| 0 < P/E < 10 | 价值被低估或夕阳行业 |
| 10 < P/E < 20 | 合理区间 |
| 20 < P/E < 30 | 成长股或高估 |
| P/E > 30 | 可能存在泡沫 |

<Aside type='caution' title='P/E使用注意' style='card'>

不同行业P/E差异巨大，银行股5倍合理，创业板30倍也合理。P/E只是参考指标之一，不能单独使用。

</Aside>

#### 市净率（P/B）

**P/B**（Price to Book Ratio，市净率）= 股价 / 每股净资产

| P/B范围 | 含义 |
|---------|------|
| P/B < 1 | 股价低于净资产，可能被低估 |
| 1 < P/B < 3 | 正常区间 |
| P/B > 3 | 相对溢价较高 |

#### 净资产收益率（ROE）

**ROE**（Return on Equity，净资产收益率）= 净利润 / 净资产

| ROE范围 | 公司评价 |
|----------|----------|
| ROE < 5% | 较差 |
| 5% < ROE < 15% | 一般 |
| 15% < ROE < 20% | 优秀 |
| ROE > 20% | 卓越 |

<Aside type='tip' title='ROE实战应用' style='card'>

持续高ROE（>15%）说明公司有护城河（竞争优势）。但要关注ROE的稳定性，持续比波动更值得关注。

</Aside>

### 2.4 仓位管理原则

**核心原则：永不满仓、永不空仓**

**仓位配置参考：**

| 投资者类型 | 股票仓位 | 现金仓位 |
|------------|----------|----------|
| 激进型 | 70-80% | 20-30% |
| 稳健型 | 50-60% | 40-50% |
| 保守型 | 30-40% | 60-70% |

**单只股票仓位上限：**
- 单只股票不超过总资金的**20%**
- 同一行业不超过总资金的**30%**
- 建议持有**5-10只**不同行业的股票

## 三、风险管理：活下来才能赚更多

### 3.1 止损是生存法则

**为什么要止损？**

假设亏损幅度与回本需要的涨幅：

| 亏损幅度 | 回本需要涨幅 |
|----------|--------------|
| -10% | +11% |
| -20% | +25% |
| -30% | +43% |
| -50% | +100% |
| -80% | +400% |

<Aside type='danger' title='止损的重要性' style='card'>

亏损50%需要翻倍才能回本！止损是保护本金最有效的手段。

</Aside>

**止损方法：**

| 方法 | 适用情况 | 优点 | 缺点 |
|------|----------|------|------|
| 固定比例止损 | 亏损8-10%出场 | 简单易执行 | 可能被正常波动扫出 |
| 固定金额止损 | 每笔交易亏2%出场 | 风险可控 | 需要计算 |
| 技术止损 | 跌破支撑位出场 | 有依据 | 需要技术分析能力 |

**建议新手用固定比例止损，8-10%是较合适的范围。**

### 3.2 新手最易犯的七大错误

1. **追涨杀跌**：看到大涨才买，大跌才卖 → 完美的高买低卖
2. **重仓单只股票**：以为集中才能赚大钱 → 实际上风险巨大
3. **忽视基本面**：只看技术面买垃圾股 → 价值可能永不回归
4. **频繁交易**：每天不交易手痒 → 手续费大量侵蚀利润
5. **杠杆交易**：借钱炒股、借信用卡炒股 → 心态崩溃的开始
6. **盲目跟风**：听消息炒股、跟大V买 → 消息到你已经晚了
7. **不止损**：亏损就死扛 → 小亏变大亏，大亏变巨亏

### 3.3 情绪管理

**情绪化交易的三个信号：**
- 🤔 "已经亏这么多了，再等等说不定反弹" → 死扛信号
- 😰 "这笔交易亏了，一定要快速赚回来" → 报复性交易信号
- 😤 "我早就知道会涨/跌，就是没买/卖" → 后视镜效应信号

<Aside type='tip' title='避免情绪化交易的方法' style='card'>

1. **制定交易计划**：买什么、买多少、什么时候买、什么时候止损
2. **写交易日记**：记录每笔交易的理由和情绪状态
3. **减少盯盘时间**：每天固定时间看盘，不随时盯
4. **保持客观**：用数据说话，不被情绪左右

</Aside>

## 四、实战入门：从模拟盘开始

### 4.1 A股交易规则

**交易时间（北京时间）：**
- 上午：9:30 - 11:30
- 下午：13:00 - 15:00

**T+1制度（当日买入股票，当日不可卖出）：**
- 当日买入的股票，**当日不能卖出**
- 卖出股票的资金**当日可用**，**次日可取**
- 这一制度**限制了日内频繁交易**

**涨跌停制度：**

| 股票类型 | 涨跌停限制 |
|----------|-----------|
| 普通**A股** | ±10% |
| **ST股票**（Special Treatment，特别处理） | ±5% |
| 科创板、创业板 | ±20% |
| 新股上市首日 | 无限制 |

**交易费用构成：**

| 费用项目 | 收取方 | 费率 |
|----------|--------|------|
| 佣金 | 券商 | 默认万2.5，可申请调低（最低万1） |
| 印花税 | 国家 | 仅卖出时收取，A股为0.05% |
| 过户费 | 交易所 | 仅沪市收取，0.001% |

### 4.2 券商开户选择

**选择券商四要素：**
1. **佣金费率**：最低万0.854（协会规定上限），主流券商在万1-万2.5
2. **软件体验**：同花顺、通达信、东方财富等第三方软件可用
3. **客户服务**：遇到问题能否及时解决
4. **线下网点**：是否方便临柜办理业务

### 4.3 模拟交易建议

<Aside type='note' title='模拟盘练习计划' style='card'>

**第1-2个月**：熟悉软件、掌握基本操作
**第3-4个月**：验证策略、记录买卖理由
**第5-6个月**：复盘总结、形成自己的交易系统

</Aside>

<Aside type='caution' title='模拟盘局限性' style='card'>

模拟资金不是真钱，心态完全不同！模拟盘赚钱不等于真钱能赚钱。请务必认识到这一点。

</Aside>

### 4.4 读财报基础

**三大财务报表：**

| 报表 | 核心问题 | 重点指标 |
|------|----------|----------|
| 资产负债表 | 公司有多少资产？欠多少债？ | 资产负债率、总资产、净资产 |
| 利润表 | 公司赚钱还是亏钱？ | 营收、净利润，毛利率 |
| 现金流量表 | 公司现金流动是否健康？ | 经营现金流 |

**必须关注的财务指标：**

| 指标 | 计算公式 | 健康范围 |
|------|----------|----------|
| 营收增长率 | (本期营收-上期营收)/上期营收 ×100% | > 10% |
| 净利润增长率 | (本期净利-上期净利)/上期净利 ×100% | > 15% |
| 毛利率 | (营收-成本)/营收 ×100% | 稳定或上升 |
| 资产负债率 | 总负债/总资产 ×100% | < 50% |
| 经营现金流 | 经营活动的现金净流量 | 正值且持续增长 |

### 4.5 个股分析框架

**第一步：行业分析**
- 行业所处周期（初创、成长、成熟、衰退）
- 行业政策环境（支持还是限制）
- 行业竞争格局（分散还是集中）

**第二步：公司分析**
- 行业地位（龙头还是跟随者）
- 竞争优势（技术壁垒，品牌、规模效应）
- 管理团队（诚信、能力）

**第三步：估值分析**
- 相对估值：P/E、P/B在行业中处于什么水平
- 绝对估值：**DCF**（Discounted Cash Flow，现金流折现）

## 五、总结语

股票投资是一门需要长期学习实践的技能，没有捷径。

<Aside type='tip' title='给新手的三个建议' style='card'>

1. **先模拟后实战**：至少模拟交易3-6个月再真钱入市
2. **先学习后投资**：先掌握基础知识再真金白银下水
3. **先止损后赚钱**：学会保护本金是盈利的前提

</Aside>

> 📈 记住：投资是一场马拉松，不是百米冲刺。活得久比赚得快更重要。
