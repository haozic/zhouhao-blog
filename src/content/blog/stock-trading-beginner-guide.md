## 一、看盘基础：如何读懂股票行情

### 1.1 K线基础

K线（Candlestick Chart）是最常用的股票技术分析工具，由**开盘价、收盘价、最高价，最低价**四个要素构成，因形似蜡烛而得名。

**K线的构成：**

- **阳线（红色/实心）**：收盘价 > 开盘价，表示上涨
- **阴线（绿色/空心）**：收盘价 < 开盘价，表示下跌
- **上下影线**：反映盘中价格的波动范围

```
     上影线
     ┌─┐
    ┌┘ └┐  ← 上影线最高点
  ──┤   ├──  ← 实体（开盘/收盘）
    └─┬─┘
     下影线
```

**常见K线形态：**

| 形态 | 特征 | 信号意义 |
|------|------|----------|
| 锤子线 | 下影线是实体的2-3倍，出现在低位 | 可能的见底信号 |
| 上吊线 | 与锤子线相反，出现在高位 | 可能的见顶信号 |
| 吞没形态 | 第二根K线完全吞没第一根 | 趋势反转信号 |
| 十字星 | 开盘=收盘，上下影线相当 | 观望信号 |

### 1.2 均线系统

移动平均线（MA, Moving Average）是最基础的趋势判断工具。

**常用均线周期：**

| 均线 | 周期 | 分析意义 |
|------|------|----------|
| MA5 | 5日 | 反映短期趋势 |
| MA20 | 20日 | 反映中期趋势 |
| MA60 | 60日 | 反映中长期趋势 |
| MA120 | 120日 | 反映长期趋势 |

**均线交叉信号：**
- **黄金交叉**：短期均线上穿长期均线 → 买入信号
- **死亡交叉**：短期均线下穿长期均线 → 卖出信号

> 💡 **实战技巧**：均线参数不是固定的。短线交易者可能用 MA5、MA10、MA20；长线投资者可能用 MA60、MA120、MA250。

### 1.3 成交量解读

**量价关系四大原则：**

| 量价关系 | 市场含义 |
|----------|----------|
| 价升量增 | 健康的多头趋势 |
| 价升量缩 | 顶背离，上涨乏力 |
| 价跌量增 | 抛压较大 |
| 价跌量缩 | 底背离，可能企稳 |

> 📝 **备注**：OBV是累计指标，需要配合其他指标一起使用才有意义。

### 1.4 常用技术指标

**MACD交易信号：**

| 信号 | 条件 | 操作 |
|------|------|------|
| 金叉 | DIF上穿DEA + MACD柱由绿转红 | 买入 |
| 死叉 | DIF下穿DEA + MACD柱由红转绿 | 卖出 |
| 底背离 | 价格创新低但MACD没有 | 可能见底 |
| 顶背离 | 价格创新高但MACD没有 | 可能见顶 |

> 💡 **布林带使用技巧**：布林带收窄往往预示行情即将突破。

## 二、交易策略入门

### 2.1 三大策略类型对比

| 策略类型 | 核心理念 | 持仓周期 | 适合人群 |
|----------|----------|----------|----------|
| 趋势跟踪 | 顺势而为 | 几周至几月 | 有时间看盘 |
| 价值投资 | 买入优质公司 | 几个月至几年 | 有耐心 |
| 短线交易 | 利用日内波动 | 分钟至几天 | 能严格止损 |

> ⚠️ **新手警告**：短线交易胜率极低（约5%），新手强烈不建议。

### 2.2 价值投资三大指标

**市盈率（P/E）= 股价 / 每股收益**

| P/E范围 | 市场判断 |
|---------|----------|
| P/E < 10 | 价值被低估 |
| 10 < P/E < 20 | 合理区间 |
| 20 < P/E < 30 | 成长股或高估 |
| P/E > 30 | 可能存在泡沫 |

> ⚠️ **注意**：不同行业P/E差异巨大，不能跨行业比较。

**净资产收益率（ROE）= 净利润 / 净资产**

| ROE范围 | 公司评价 |
|----------|----------|
| ROE < 5% | 较差 |
| 5% < ROE < 15% | 一般 |
| 15% < ROE < 20% | 优秀 |
| ROE > 20% | 卓越 |

> 💡 持续高ROE（>15%）说明公司有护城河竞争优势。

## 三、风险管理

### 3.1 止损是生存法则

| 亏损幅度 | 回本需要涨幅 |
|----------|--------------|
| -10% | +11% |
| -20% | +25% |
| -30% | +43% |
| -50% | +100% |
| -80% | +400% |

> 🔴 **核心提醒**：📉 亏损50%需要翻倍才能回本！止损是保护本金最有效的手段。

**建议新手用固定比例止损，8-10%是较合适的范围。**

### 3.2 新手最易犯的七大错误

1. **追涨杀跌** → 高买低卖
2. **重仓单只股票** → 风险巨大
3. **忽视基本面** → 价值可能永不回归
4. **频繁交易** → 手续费大量侵蚀利润
5. **杠杆交易** → 心态崩溃的开始
6. **盲目跟风** → 消息到你已经晚了
7. **不止损** → 小亏变大亏

> ⚠️ **散户亏损最主要的原因**："不止损"是散户亏损的最主要原因。

## 四、实战入门

### 4.1 A股交易规则

**交易时间（北京时间）：**
- 上午：9:30 - 11:30
- 下午：13:00 - 15:00

**T+1制度：** 当日买入的股票，当日不能卖出。

> ⚠️ **T+1注意**：当日买入后下跌，无法当日止损，只能等到下一个交易日。

**涨跌停限制：**
- 普通A股：±10%
- ST股票：±5%
- 科创板、创业板：±20%

### 4.2 读财报基础

**必须关注的财务指标：**

| 指标 | 健康范围 |
|------|----------|
| 营收增长率 | > 10% |
| 净利润增长率 | > 15% |
| 毛利率 | 稳定或上升 |
| 资产负债率 | < 50% |
| 经营现金流 | 正值且增长 |

**财务排雷信号：**
- 🔴 营收增长但经营现金流持续为负
- 🔴 毛利率持续下降
- 🔴 应收账款大幅增长

## 五、总结语

**给新手的三个建议：**

1. **先模拟后实战**：至少模拟交易3-6个月再真钱入市
2. **先学习后投资**：先掌握基础知识再真金白银下水
3. **先止损后赚钱**：学会保护本金是盈利的前提

> 💡 **投资箴言**：📈 投资是一场马拉松，不是百米冲刺。活得久比赚得快更重要。
